## 🔍 자료형

### 1️⃣ 수 자료형

1. **정수형** 
    - 양수, 정수, 0
2. **실수형** 
    - 소수부나 정수부가 0일 때 생략 가능 ex) .7, -5.
    - e(E) 표기법:
        - 유효숫자e^(지수) = 유효숫자 x 10^(지수) ⇒ 자릿수 실수를 할 확률을 줄임
        - 무한(INF)를 1e9(10억)로 표현하기도 함
    - round(실수, 반올림할 위치 - 1):
        - round가 필요한 이유? - 보통 소수점 다섯번째 자리에서 반올림한 결과가 같으면 정답으로 인정
            
            ```python
            a = 0.3 + 0.6
            print(a)
            ====
            0.899999999....  #2진수로 0.9를 정확히 표현할 방법이 없음
            ```
            
        - [ round vs print(’%.f’, %_) ]
            - round(1.4736, 2) ⇒ 1.5 (0은 생략 됨)
            - print(’%.2f’,  %실수변수)에 1.4736을 넣으면? ⇒ 1.50
3. **연산** 
    - 코드를 짤 때 0은 항상 예외적으로 검사하기
    - 일반 연산자: +, -, *,  /(일반적인 나누기), //(몫), %(나머지), **(거듭제곱)
    - 쉬프트 연산자: << N (2^N배), >> N ((1/2)^N배)
    - 비트단위 연산자: ~(not), &(and), |(or), ^(nor)
    

### 2️⃣ 리스트 자료형

1. **소개**
    - 연결 리스트 자료 구조 기능 포함 → append(), remove() 등의 메서드 지원
    - C++ STL vector와 유사
2. **리스트 생성**
    - list() or []
    - 1차원 배열 초기화: arr = [0] * N
    - 2차원 배열 초기화: arr = [[0] * N for _ in range(M)]
3. **인덱싱과 슬라이싱**
    - 인덱스는 음,양의 정수 모두 가능.
    - 음의 정수의 경우 거꾸로 탐색
    - arr[a:b] ⇒ a부터 b-1 까지 가져오기
4. **리스트 컴프리헨션 (예시)**
    1. array = [i for i in range(20) if i % 2 == 1]  ⇒ 0~19까지의 수 중 홀수만 포함하는 리스트 생성
    2. array = [i * i for i in range(1,10)]  ⇒ 1~9까지의 수의 제곱 값을 포함하는 리스트
    3. 2차원 배열 초기화 방법도 여기에 해당
5. **메서드**
    1. append(): 원소 하나 삽입
    2. sort(): 오름차순 정렬. reverse = True 인자를 넣어주면 내림차순 정렬
    3. reverse(): 원소의 순서를 모두 뒤집기
    4. insert(삽입할 인덱스, 값): 특정위치에 원소 삽입
    5. count(값): 특정 값을 가지는 데이터 개수 세기
    6. remove(값): 특정 값을 가지는 원소 제거. 여러개가 있으면 하나만 제거한다.
