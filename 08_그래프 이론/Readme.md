# 💡 다양한 그래프 알고리즘

1. DFS/BFS와 최단 경로 모두 그래프 알고리즘의 한 유형
2. 이외에도 다양한 그래프 알고리즘 존재(출제 빈도 ⬇️, but 알아두기) 
- 서로소 집합
- 크루스칼
- 위상정렬
3. "서로 다른 개체가 연결되어 있다" => 그래프 알고리즘 떠올리기
4. 그래프 vs 트리( = 그래프 자료구조의 일부)

    | |  그래프  |  트리 
    | - | ----| ----|
    방향성| 방향 혹은 무방향 | 방향 (일반적인 수학에서는 무방향으로 간주)
    순환성 | 순환 및 비순환 | 비순환
    루트 노드 존재 여부| 없음 | 있음
    부모자식 관계 | 없음 | 있음
    모델의 종류 | 네트워크 모델 | 계층 모델
